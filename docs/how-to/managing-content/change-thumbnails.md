# Changing a thumbnail

Sometimes you might want to change the generated thumbnail for an item to a custom image, or change the representative image for a collection. This is the right way to do it.

## Changing the Collection thumbnail (Representative Object)

Because of our theming, Collection item thumbnails work a little differently from other repository content. To change the Collection thumbnail:

  - Edit the collection
  ![<# The Collection header, showing representative object and Edit button #>](/arca-docs/assets/collection-rep-image.png "Screenshot")
  - On the metadata form, scroll down to the Representative Image section, and expand it
  ![<# The expanded Representative Image section on the metadata form #>](/arca-docs/assets/rep-image-delete.png "Screenshot")
  - Click the trash button on the image you no longer want to use, to remove it
  - Click "Select Entities" to pick a new piece of media to use as your image
  - Either search for an existing image to use, or upload a new one
  ![<# The Select Entities screen showing how to select new media for the collection #>](/arca-docs/assets/rep-image-replace.png "Screenshot")  
  - Save your work
  
## Changing Repository Item thumbnails

While Collections use the Representative Image, thumbnails for normal repository content (images, newspapers, videos, theses, etc.) are found on the item's Media tab.

  - Navigate to the item, and click the Media tab
    ![<# The Media tab for a Repository Item #>](/arca-docs/assets/manage-media.png "Screenshot")  
  - Find the Media item with Media Use "Thumbnail Image". Click the **EDIT button** (NOT the delete button)
    ![<# The Thumbnail Image media item, with Edit button highlighted #>](/arca-docs/assets/manage-thumbnail.png "Screenshot")  
  - On the Edit screen, find the Image field, and click the Remove button
     ![<# The Image field on a Thumbnail media item #>](/arca-docs/assets/thumbnail-image-remove.png "Screenshot")   
  - Click the "Add new file" button that just appeared, and upload your new thumbnail
     ![<# The "add new file" interface for the Image field #>](/arca-docs/assets/thumbnail-image-upload.png "Screenshot")     
  - Save
